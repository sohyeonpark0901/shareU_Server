## Service 

1. 포인트

- 신규 회원 가입시 포인트 지급
- 포인트 적립/차감을 통해 자료 공유와 다운로드 가능
- 별점과 리뷰 등록시 포인트 지급


2. 자료 공유와 다운로드

- 다양한 사용자가 자료 공유와 다운로드 가능


3. 리뷰 / 별점 

- 리뷰와 별점을 자유롭게 등록
- 일반 사용자도 리뷰와 별점을 자유롭게 조회 가능
- 리뷰 / 별점 순으로 자료를 나열하여 더좋은 자료를 검색할 수 있도록 유도

4. 다양한 검색
- 자료명 또는 학과 검색을 통해 자료 리스트 조회

5. 설정

- 사용자 정보 변경 설정
